# Industry Analyzer Portfolio Demo

A deployable Next.js demo with full user journey:
- Industry and template selection
- CSV upload + validation
- Auto visualizations
- AI-generated insights + chatbot via OpenRouter
- Export/share actions

## Setup
1. Install Node.js 20+.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy env file:
   ```bash
   cp .env.example .env.local
   ```
4. Fill `.env.local` with your keys.
5. Run:
   ```bash
   npm run dev
   ```

## Supabase (optional but included)
Create table:
```sql
create table if not exists uploaded_records (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default now(),
  industry text,
  template_id text,
  payload jsonb
);
```

## OpenRouter Model
The API route uses a free model:
- `meta-llama/llama-3.3-8b-instruct:free`

You can change model in:
- `src/app/api/ai/route.ts`

## Deploy
- Push to GitHub
- Import repo into Vercel
- Add env vars in Vercel Project Settings
- Deploy

## Security
Do not commit `.env.local`.
If your key was exposed publicly, rotate it immediately in OpenRouter.
